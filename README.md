## 🚀 **Quick Start** (< 3 minutes)

```bash
# 1. Clone & Install
git clone https://github.com/Gzeu/protogx-network-suite.git
cd protogx-network-suite
npm run setup

# 2. Environment
cp .env.example .env.local
# Edit .env.local with your API keys

# 3. Development
npm run dev
# Open http://localhost:3000

# 4. Deploy (Optional)
npm run deploy
```

## 🔧 **Commands**

```bash
# Development
npm run dev          # Start with Turbo
npm run build        # Production build
npm run test         # Unit tests
npm run e2e          # E2E tests

# Analysis
npm run analyze      # Bundle analyzer
npm run performance  # Lighthouse
npm run lint         # ESLint + fix

# Blockchain
npm run contracts:build   # Smart contracts
npm run contracts:deploy  # Deploy testnet

# Database
npm run db:generate  # Supabase types
npm run db:migrate   # Migrations

# Deployment
npm run deploy       # Vercel production
```
